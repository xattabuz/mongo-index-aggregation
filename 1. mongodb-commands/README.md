<h1>1. Database</h1>

<br>
<h3>
	1.1 Show database  
</h3>
<br>

Within the shell,  refers to your current database. Type `show database` or `db`. to display the current database.
<br>

```
show databases
```

<br>
<h3>
	1.2 Switch database 
</h3>
<br>

To switch databases, type `use db`. For example, to switch to the examples database:

```
use example_db
```
